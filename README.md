# solUSD

Manna Finance allows users to mint solUSD with SOL as collateral.

Main Functionalities:

- Open Trove
- Deposit Collateral
- Withdraw Collateral
- Withdraw solUSD
- Repay solUSD
- Adjust Trove
- Close Trove
- Claim Collateral

## Development

build:

```bash
anchor build
```

test:

```bash
anchor build
```

deploy:

```bash
anchor deploy
```
